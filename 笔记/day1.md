### 1.初始化创建此项目，并依次创建此项目需要用到的模块化管理文件夹 并进行第一次提交代码至 github 上

### 2.配置别名路径联想提示 顾名思义：就是方便以后导包时不用输入太长的路径

在项目根目录下新建 jsconfig.json 文件
相关配置代码如下：

```json
{
  "compilerOptions": {
    "baseUrl": "./",
    "paths": {
      "@/*": ["src/*"]
    }
  }
}
```

只是做了一个联想提示 实际路径转换在配置文件中
